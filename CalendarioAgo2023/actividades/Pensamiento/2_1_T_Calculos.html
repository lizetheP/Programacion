    • Solicite al usuario el valor asignado de los lados B y C de un triángulo rectángulo y calcule el valor resultante del lado A. Para la solución de este problema se sugiere que utilices el teorema de Pitágoras. El cálculo de la raíz cuadrada se puede realizar utilizando la función math.sqrt() perteneciente a la libreria math (import math). Guarda el archivo con el nombre L1_Matricula.py

Por ejemplo:

x = math.sqrt(y)      La raiz de y se guarda en la variable x
x = math.sqrt(y+z)  La raiz de y + z se guarda en la variable x.
