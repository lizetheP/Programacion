TC1027. Programación para negocios 
.  

Instalación de librerías
.
.

   Objetivo:  
    • Instalar las librerías: numpy, pandas, matplotlib

Modalidad:  
    • Actividad individual.

 
  Instrucciones: 


    • Instala las siguientes librerías:

      • numpy
      • pandas
      • matplotlib
      • xlrd
      • openpyxl

    • En Tools > Manage packages...

                     

    • Escribimos el nombre de la librería, por ejemplo: pandas, da clic en el botón Search on PyPI
    • En Search results da clic en la librería a instalar, por ejemplo: pandas

                     

    • En la ventana que aparece da clic en Install

                 

    • Repite el mismo proceso para todas las librerías a instalar.

                 

    • En Thonny, crea un archivo llamado prueba.py
    • Guarda el archivo vendedores.xlsx en la misma carpeta donde guardaste tu archivo prueba.py
    • En tu archivo prueba.py copia el siguiente código y ejecútalo:

      • import pandas as pd
      import numpy as np
      import matplotlib.pyplot as plt
      import pandas as pd
      import statistics
      import openpyxl

      tabla = pd.read_excel("vendedores.xlsx")
      print(tabla)
    • Realiza una impresión de pantalla de tu ejecución en el Shell.

                             

    • En un documento de word, copia la impresión de pantalla y guarda tu archivo como Instalado.docx

 Especificaciones de entrega:
 
    • Formato de entrega: docx o pdf
    • Nombre de los entregables: instalado.docx
    • Medio de entrega: Se entrega en Instalación de librerías
